# Admin CLI

Инструмент командной строки для администрирования AthleticaCRM.

## Назначение

Admin CLI предназначен для выполнения административных задач напрямую через базу данных, без необходимости запускать основное приложение. Используется для:

- Поиска организаций по названию или логину владельца
- Просмотра информации об организациях
- Корректировки балансов организаций

## Установка и запуск

### Переменные окружения

Для работы требуются переменные окружения для подключения к базе данных:

- `POSTGRES_URL` — JDBC URL подключения (например, `jdbc:postgresql://localhost:5432/athletica`)
- `POSTGRES_USER` — имя пользователя
- `POSTGRES_PASSWORD` — пароль

### Запуск через скрипт

```bash
./admin-cli/build/install/admin-cli/bin/admin-cli [команда] [опции]
```

### Запуск через JAR

```bash
java -jar admin-cli/build/libs/admin-cli-1.0.0-all.jar [команда] [опции]
```

## Команды

### Поиск организаций

Поиск по названию организации или логину владельца:

```bash
admin-cli --find=<запрос>
```

Примеры:

```bash
# Поиск по названию
admin-cli --find=ЦР

# Поиск по логину владельца
admin-cli --find=admin

# Несколько запросов одновременно
admin-cli --find=ЦР --find=admin
```

Вывод:

```
Результаты для "ЦР":
  019e1c25-7d23-7234-9ef8-160057bc67e4  ЦРФКС  (владелец: admin, валюта: RUB)
```

### Корректировка баланса организации

Зачисление или списание средств на баланс организации:

```bash
admin-cli --org=<UUID или название> --credit=<сумма> --description=<описание>
admin-cli --org=<UUID или название> --debit=<сумма> --description=<описание>
```

Параметры:

- `--org` — UUID организации или название (если название неоднозначно, будет предложен список для выбора)
- `--credit` — сумма зачисления в основных единицах валюты (рубли, доллары и т.д.)
- `--debit` — сумма списания в основных единицах валюты
- `--description` — описание операции (обязательно)

Примеры:

```bash
# Зачисление 1000 рублей
admin-cli --org=ЦРФКС --credit=1000 --description="Тестовое зачисление"

# Списание 500 рублей по UUID
admin-cli --org=019e1c25-7d23-7234-9ef8-160057bc67e4 --debit=500 --description="Возврат средств"
```

Вывод:

```
Организация: ЦРФКС (019e1c25-7d23-7234-9ef8-160057bc67e4)
Корректировка: +1 000,00 ₽
Новый баланс:  15 000,00 ₽
```

## Сборка

```bash
# Сборка JAR
./gradlew :admin-cli:shadowJar

# Сборка дистрибутива со скриптами запуска
./gradlew :admin-cli:installDist
```

## Требования

- Java 21+
- PostgreSQL с базой данных AthleticaCRM
